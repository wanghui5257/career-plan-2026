# 职业发展计划 2026 💼

个人计划管理 Web 应用 - 助力职业转型 AI 领域

## 项目目标

- 📊 可视化任务管理
- 📅 每日计划追踪
- 📈 进度可视化
- 🔄 反馈循环

## 技术栈

### 前端
- React / Vue (待定)
- 任务板界面
- 图表展示

### 后端
- Java 17
- Spring Boot 3.x
- Spring Security (JWT)
- MySQL / PostgreSQL

### 部署
- 阿里云 ECS
- Docker
- Nginx

## 快速开始

```bash
# 后端
cd backend
./mvnw spring-boot:run

# 前端
cd frontend
npm install
npm run dev
```
